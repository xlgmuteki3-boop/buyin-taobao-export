# 百应+淘宝 商品一键导出

用于在巨量百应、淘宝、天猫页面抓取商品数据，并支持导出 CSV 或写入飞书多维表格。

## 安装方式

### 1. 安装 Tampermonkey
请先在浏览器安装 Tampermonkey 扩展。

### 2. 安装脚本
打开下面链接后点击安装：

https://raw.githubusercontent.com/xlgmuteki3-boop/buyin-taobao-export/main/buyin_taobao_export.user.js

## 适用页面

- `https://buyin.jinritemai.com/*`
- `https://*.buyin.jinritemai.com/*`
- `https://*.taobao.com/*`
- `https://*.world.taobao.com/*`
- `https://*.tmall.com/*`
- `https://*.tmall.hk/*`

## 功能说明

1. 抓取百应商品数据
2. 抓取淘宝/天猫商品数据
3. 支持导出 CSV
4. 支持写入飞书多维表格
5. 支持字段映射
6. 支持自动建列
